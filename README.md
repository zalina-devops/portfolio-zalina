## 🧠 Концепция

**Portfolio Zalina** — это не обычный сайт-портфолио. Это интерактивная терминальная среда, стилизованная под хакерский интерфейс. Посетитель взаимодействует через командную строку, исследуя навыки, проекты и путь развития в формате настоящего DevOps-окружения.


---

## ✨ Возможности

| Фича | Описание |
|------|----------|
| 🖥️ **Терминальный интерфейс** | Полноценный CLI с историей команд и автодополнением |
| ⌨️ **Звуки механической клавиатуры** | Реалистичные звуки нажатий через Web Audio API |
| 🧠 **Нейросетевой фон** | Частицы соединяются линиями, реагируют на мышь |
| 🌧️ **Matrix Rain** | Пасхалка — цифровой дождь по команде `matrix` |
| 🎮 **Игра «Hack the Server»** | Взломай 4-значный код за 8 попыток + салют при победе |
| 📄 **PDF-резюме** | Скачивание резюме по команде `resume` |
| 🪟 **Многооконность** | Перетаскиваемые окна как в настоящей ОС |
| 💥 **Глитч-эффекты** | Экран «ломается» при неверных командах |
| 📦 **GitHub API** | Живые данные о репозиториях |
| 🎨 **Глассморфизм** | Современный дизайн с размытием и прозрачностью |
| 📱 **Адаптивность** | Корректное отображение на мобильных устройствах |

## 📟 Команды терминала

| Команда | Описание |
|---------|----------|
| `help` | Список всех команд |
| `whoami` | Информация обо мне |
| `skills` | Технические навыки |
| `projects` | Проекты с GitHub (live) |
| `education` | Образование и планы |
| `timeline` | Мой путь развития |
| `contact` | Контакты |
| `resume` | Скачать PDF-резюме |
| `matrix` | Пасхалка (Matrix Rain) |
| `game` | Игра «Взлом сервера» |
| `stats` | Статистика портфолио |
| `coffee` | Экстренная заправка |
| `sudo` | Попробуй ;) |
| `clear` | Очистить терминал |

## 🚀 Быстрый старт

```bash
# Клонировать репозиторий
git clone https://github.com/zalina-devops/portfolio-zalina.git

# Перейти в папку
cd portfolio-zalina

# Установить зависимости
npm install

# Запустить локально
npm run dev
```
